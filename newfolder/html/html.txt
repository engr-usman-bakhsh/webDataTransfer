36-Summary of Chapter4

>> to create hyperlink in HTML we use anchor tag, <a></a>
>>the text between opening <a> tag and closing </a> tag is called the Link Text
>>  there are two basic choices when it comes to the selection of URLs,
    -- Absolute URL. usually used to navigate to another site.
    -- Relative URL. usually used to navigate to the same website
>> Anchor element always carry an attribute called href(hyperlink refference). The value of the href attribute is the URL that you want people to go to when they click  on the link text.( the value of href attribuute becomes a link)
>> Either relative or absolute URLs may be used inside the href attribute.
  -- when linking to a different website , the value of the href attribute will be the full web address for the website (including the domain name). this is the absolute URL
  -- when youre linking to other pages within the same site, you can use relative URLs. Relative URLs are like a short-hand version of Absolute URLs because there is no need to specify the domain name.
  -- relative URLs help when building a site on your computers because you can create links between pages without having to setup your domain name or hosting.
>> to open a link in new browser window or tab you can use the target attribute in the opening <a> tag. thr value of the target attribute should be "_blank".
>> the default value of the target attribute is "_self" which means link will open in the same window.

55- Forms Structure.

form control means collecting information from users. we make basic form structure from form elements. all form control are called  inside the <form> element.  <form> element always carry  "Action" attribute and its "value" is a "url" where we have to submit our forms. "Method" attribute is also used. form is filled at client side and we send forms to server by using 2 methods. these methods are "get method" or "post method". if we donot specify the method attribute than the default method is "get method". we can say the "method" attribute can have 2 values. "get" or "post". "get" method is used when form length is short. by using this method our all sent information will be displayed at the end of URL. "post" method is used to send sensitive data.

When to use the GET method?
>> with the GET method, the length of a URL is limited(2048 characters)
>> Never use GET method to send sensitive data( data will be visible in the URL)
>> GET method is better for non-secure data, like query strings in Google.
-- GET method is just like the searching on google. our surched keywords are displayed in the URL.


When to use "POST" method?
>> POST has no size limitations and can be used to send large amount of data.

   -- Always use POST if the form data containes sensitive or personal information.( the post method does not display the submitted data in the page address field)
   -- POSt has no size limitation and and can be used to send large amount of data
   -- form submission with POST can not be bookmarked.
   -- Data from a form is sent as name/value pairs.
   -- Form controls may be used to collect data from the users. each form control is given a name and the text the user types in or the values of the option they
